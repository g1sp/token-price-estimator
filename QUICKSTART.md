# Quick Start - 30 Seconds

## Run Locally

### Option 1: Just Open the File
```bash
open index.html
# or double-click index.html in Finder/Explorer
```

### Option 2: Use a Server (Recommended)
```bash
# Python 3
python -m http.server 8000

# or Python 2
python -m SimpleHTTPServer 8000

# or Node.js
npx http-server
```

Then visit: **http://localhost:8000**

---

## Use It

1. **Select Provider** → Choose Claude, Gemini, or OpenAI
2. **Paste Plan** → Copy your project plan into the textarea
3. **Pick Scenario** → Choose Summary, Code, or Analysis
4. **Calculate** → Click "Calculate Tokens"
5. **Copy Results** → Click "Copy Results" button

---

## Deploy to GitHub Pages

```bash
# 1. Create a new GitHub repo called: token-price-estimator
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR-USERNAME/token-price-estimator.git
git branch -M main
git push -u origin main

# 2. Go to Settings → Pages → Main branch → Save
# 3. Wait 1-2 minutes
# 4. Live at: https://YOUR-USERNAME.github.io/token-price-estimator
```
